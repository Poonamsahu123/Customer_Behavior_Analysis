# 📊 Customer Shopping Behavior Analysis

## 🔍 Overview

This project focuses on analyzing customer shopping behavior using a combination of **Python, SQL, and Power BI**.
The goal is to extract meaningful insights from raw data by performing data cleaning, exploratory data analysis (EDA), and querying structured data using SQL.

---

## 📁 Dataset

The dataset contains information about customer purchases, including:

* Customer demographics (age, gender)
* Product categories
* Purchase amount
* Discount usage
* Ratings

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **PostgreSQL / MySQL / SQL Server**
* **SQL** (for querying and analysis)
* **Power BI** (for dashboard creation)
* **Jupyter Notebook**
* **Git & GitHub**

---

## ⚙️ Project Workflow

### 1. Data Loading

* Loaded dataset using Python (Pandas)
* Inspected structure using `.info()` and `.describe()`

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types where necessary

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Visualized purchase trends
* Identified top categories and spending patterns

### 4. SQL Analysis

* Imported dataset into PostgreSQL
* Performed queries such as:

  * Revenue by gender
  * Customers using discounts
  * Average purchase analysis
  * Product performance

### 5. Dashboard (Power BI)

* Created interactive dashboard
* Visualized:

  * Revenue distribution
  * Category performance
  * Customer segmentation

### 6. Reporting

* Summarized key insights
* Created presentation (PPT using Gamma)

---

## 📊 Key Insights

* Identified top-spending customer segments
* Found impact of discounts on purchase behavior
* Highlighted best-performing product categories
* Analyzed customer age group trends
